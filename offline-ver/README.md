# Взлом Перикота

Автономная мобильная HTML-версия мини-игры про взлом терминала через сопоставление фигур.

## Запуск

Откройте файл `index.html` на телефоне. Интернет, сервер, `npm install` и сборка не нужны.

## Что внутри

- Вся логика, стили и разметка находятся в одном `index.html`.
- Внешних шрифтов, CDN, картинок и модульных скриптов нет.
- Звуки синтезируются через Web Audio API после касания экрана.
- Управление рассчитано на телефон: кнопки и свайп влево/вправо по буферу модулей.

## Правила

Нужно выбрать модуль, который закрывает пустые клетки целевой матрицы и не попадает на белые клетки. После трех успешных синхронизаций протокол считается взломанным.
